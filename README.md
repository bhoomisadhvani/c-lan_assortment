📘 C Programs – Array Operations

This repository contains basic C programs related to array operations including:

Finding negative elements in a 1D array

Finding the largest element in a 2D array

Transpose of a matrix

🧾 Programs Included
1️⃣ Find Negative Elements in Array

Description:
This program takes input for an array and prints all negative elements.

Steps:

Enter size of array

Input elements

Loop through array

Print elements less than 0

Example Output:

Enter array size: 5
a[0] = 10
a[1] = -5
a[2] = 3
a[3] = -2
a[4] = 7

Negative elements: -5 -2
2️⃣ Largest Element in 2D Array

Description:
This program finds the largest element in a matrix.

Steps:

Enter number of rows and columns

Input matrix elements

Compare each element

Display maximum value

Example Output:

Enter row size: 2
Enter column size: 2

a[0][0] = 1
a[0][1] = 5
a[1][0] = 3
a[1][1] = 9

Largest element is: 9
3️⃣ Transpose of Matrix

Description:
This program calculates the transpose of a square matrix.

Steps:

Enter size (n x n)

Input elements

Swap rows and columns

Print transpose

Example Output:

Enter row & column size: 2

a[0][0] = 1
a[0][1] = 2
a[1][0] = 3
a[1][1] = 4

The transpose matrix:
1 3
2 4
